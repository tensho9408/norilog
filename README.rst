============
乗ログアプリ
============

目的
====


Webブラウザーでコメント投稿するWebアプリケーションの練習

ツールのバージョン
=================

:Python:        3.6.4
:pip:           10.0.1

インストールと起動方法
=====================

 $ git clone https://github.com/tensho9408/norilog.git
 $ cd norilog 
 $ python3.6 -m venv venv
 $ source venv/bin/activate
 (venv) $ pip install .
 (venv) $ norilog
 * Runing on httpp://127.0.0.1:8000/

開発手順
========

開発用インストール
==================


1.チェックアウトする
2.以下の手順でインストール::

    (venv) $ pip install -e .
